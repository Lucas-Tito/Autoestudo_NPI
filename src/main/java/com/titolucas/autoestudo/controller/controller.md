Controller faz parte de uma organização conhecida como arquitetura em camadas.

Ela ajuda a separar responsabilidades, deixar o código mais limpo, organizado e fácil de manter.

controller – Ponto de entrada da API (REST)

    Recebe as requisições HTTP (GET, POST, PUT, DELETE).

    Conversa com o service e devolve uma resposta para o cliente.

    Basicamente: traduz a API externa pro funcionamento interno da sua aplicação.

Garçom – recebe o pedido e entrega o prato
